Bifrost
Welcome to Bifrost! This tool allows you to export components from anywhere you see on the web, and import it into Figma.
In order to use this tool:
  • Click the below button to start inspection
  • Click any HTML element
  • The element's styles will be retrieved and converted into JSON for Figma integration.
